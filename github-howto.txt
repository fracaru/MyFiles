Global setup:

 Set up git
  git config --global user.name "username"
  git config --global user.email john@doe.com
      

Next steps:

  mkdir MyFiles
  cd MyFiles
  git init
  touch README
  git add README
  git commit -m 'first commit'
  git remote add origin git@github.com:username/MyFiles.git
  git push -u origin master
      

Existing Git Repo?

  cd existing_git_repo
  git remote add origin git@github.com:username/MyFiles.git
  git push -u origin master
